## 🆘 Suporte Rápido

### Problema: Página não carrega corretamente
✅ Verificar se todas as imagens/recursos estão no mesmo servidor  
✅ Verificar permissões dos arquivos (644)  
✅ Limpar cache do navegador

### Problema: Formulário não envia
✅ Verificar se o backend do formulário está configurado  
✅ Testar em navegador diferente  
✅ Verificar console do navegador (F12) para erros

### Problema: Métricas não aparecem no GA
✅ Confirmar que o código do GA está no HTML  
✅ Aguardar 24-48h para dados aparecerem  
✅ Usar modo debug do GA para testar
